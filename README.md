# RG Seguranca

This project is a security management system built with React, Node.js, and PostgreSQL.

## Getting Started

### Prerequisites
- Node.js
- PostgreSQL

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/<YOUR_USERNAME>/rg-seguranca-react.git
   ```
2. Install dependencies
   ```bash
   cd rg-seguranca-react
   npm install
   ```

### Running the App

- For Frontend:
  ```bash
  cd frontend
  npm start
  ```

- For Backend:
  ```bash
  cd backend
  npm start
  ```
